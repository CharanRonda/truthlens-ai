# ThruthLens AI

ThruthLens AI is a full-stack interview intelligence demo built around your reference `in.html` design. The frontend mirrors that dark, data-driven UI while the backend exposes mock real-time analysis endpoints that simulate:

- Facial emotion detection
- Voice stress analysis
- Behavior tracking
- Environment monitoring

## Stack

- Frontend: React + Vite + Chart.js + jsPDF
- Backend: FastAPI
- AI integration points: OpenCV, DeepFace, Librosa, YOLO

## Project Structure

- `frontend/` React application
- `backend/` FastAPI application

## Run The Backend

```bash
cd /Users/charanronda/Documents/New\ project/backend
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## Run The Frontend

```bash
cd /Users/charanronda/Documents/New\ project/frontend
npm install
npm run dev
```

The Vite dev server proxies `/api/*` requests to `http://127.0.0.1:8000`.

## Current Behavior

- Candidate sessions use the local webcam on the frontend.
- Analysis data is mock-generated by FastAPI to simulate a live AI pipeline.
- The PDF report follows the same structure and feel as the reference HTML file, while expanding the content to include facial, voice, behavior, and environment sections.

## Where Real AI Models Plug In

Replace the mock generators in [backend/app/engine.py](/Users/charanronda/Documents/New project/backend/app/engine.py) with:

- OpenCV or DeepFace for emotion and face analysis
- Librosa for voice stress and audio feature extraction
- YOLO for phone, multiple-person, and suspicious-object detection

The frontend data contract is already shaped for that upgrade, so you can swap the mock layer without redesigning the UI.
